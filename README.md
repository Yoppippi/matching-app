# 利用方法
 
* 画面中央にスタートボタンが二つありますので、上から順に行ってください。
* 一つ目は簡単な性格診断テストです。
* 二つ目は好きなものアンケートです。
* 二つとも終えましたら、下に結果と「あなたにおすすめ」が表示されます。
 
# 目指した課題解決
 
 * おすすめの○○を手軽に表示する
 * 新しい自分に気付けるきっかけを作る
 * 新しい発見を見つける

# 要件
 
* 「あなたにおすすめ」を手軽に見やすく表示させる
